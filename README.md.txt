# 📡 Smart Telecom Tower Monitoring System

A real-time IoT monitoring system built using ESP32, MQTT, and Node-RED.

## Features

- Real-time Battery Voltage Monitoring
- Temperature Monitoring
- Signal Strength (RSSI)
- Live Dashboard
- MQTT Communication
- Tower Status Indicator
- RSSI History Graph

## Technologies

- ESP32
- MQTT
- Node-RED
- Wokwi
- Arduino IDE
